# Sitio Educativo

Sitio educativo estatico listo para publicar en GitHub y Render.

## Paginas

- `index.html`: home
- `cursos.html`: catalogo de cursos
- `curso-detalle.html?curso=gestion-empresarial`: detalle de curso
- `login.html`: acceso de usuario
- `aula.html?curso=gestion-empresarial`: aula del curso comprado

## Render

Este proyecto incluye `render.yaml` para publicarlo como Static Site.

- Build command: `echo "Static site ready"`
- Publish directory: `.`
